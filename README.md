# Bantay Presyo API

A REST API for accessing the Philippine Agricultural products market price

## Installation

```bash
git clone git@github.com:wjamesdevera/bantay-presyo-api.git
```

## Usage

```python
pip install -r requirements.txt
fastapi run app/main.py

```
